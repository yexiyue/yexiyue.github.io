# 4.rn样式

StyleSheet · React Native 中文网 (opens new window)

react native中使用styleSheet api进行样式编写

1.与css的区别

  • 没有继承性（Text组件除外）
  • 样式名采用小驼峰命名
  • 所有尺寸都是没有单位
  • 有些特殊的样式名（例如：marginHorizontal）

2.样式声明

通过style属性直接声明

  • 属性值为对象: <组件style={ {样式} }/>
  • 属性值为数组: <组件 style={ [ {样式1}, ... {样式N}]}/>

在style属性中调用StyleSheet声明的样式

  1. 引入: import { StyleSheet, View } from 'react- native'
  2. 声明: const styles = StyleSheet.create({ foo:{样式1}, bar: {样式2} })
  3. 使用: <View style={ [ styles.foo, styles.bar ] } >内容</View>

# 5.组合动画

动画还可以使用组合函数以复杂的方式进行组合：

  • Animated.delay() (opens new window)在给定延迟后开始动画。
  • Animated.parallel() (opens new window)同时启动多个动画。
  • Animated.sequence() (opens new window)按顺序启动动画，等待每一个动画完成后再开始下一个动画。
  • Animated.stagger() (opens new window)按照给定的延时间隔，顺序并行的启动动画。

动画也可以通过将toValue设置为另一个动画的Animated.Value来简单的链接在一起。请参阅动画指南中的跟踪动态值 (opens new window)值。

默认情况下，如果一个动画停止或中断，则组合中的所有其他动画也会停止。
